# 🧠 CLI Resume Application
The CLI Resume Application is a command-line interface-based resume builder that allows users to create and display their resumes in a visually appealing way. The application uses a combination of libraries such as `inquirer`, `chalk`, and `terminal-link` to provide an interactive and engaging experience. The core features of the application include displaying a welcome message and animation, handling user interactions, and displaying resume information.

## Installation
- Run `npm i -g shubham0401`
- Then in terminal run `shubham0401`

## 🚀 Features
- Interactive resume builder using `inquirer` for user input handling
- Visually appealing welcome message and animation using `cfonts` and `chalk-animation`
- Display of resume information with clickable links for contact information using `terminal-link`
- Customizable and reusable animation and welcome message functions
- Easy-to-use command-line interface for users to navigate and display their resumes

## 🛠️ Tech Stack
* Frontend: `inquirer` for user input handling, `chalk` for text styling, `terminal-link` for creating clickable links
* Backend: None
* Database: `./data/resume.json` for storing resume data
* AI Tools: None
* Build Tools: `eslint` and `eslint-plugin-node` for linting and testing
* Libraries: `cfonts`, `chalk`, `chalk-animation`, `inquirer`, and `terminal-link`

## 📦 Installation
To install the application, follow these steps:
1. Clone the repository using `git clone`
2. Install the dependencies using `npm install`
3. Run the application using `npm start`

## 💻 Usage
To use the application, simply run `npm start` and follow the prompts to display your resume.
